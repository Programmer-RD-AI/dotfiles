# Tmux Configuration

## Key Bindings

**Prefix**: `Ctrl+a`

**Panes**
- `Ctrl+a |` → Split vertical
- `Ctrl+a -` → Split horizontal
- `Alt+Arrow` → Navigate panes (no prefix)
- `Ctrl+a Arrow` → Resize panes

**Copy Mode**
- `Ctrl+a [` → Enter copy mode
- `v` → Begin selection
- `y` → Copy and exit

**Config**
- `Ctrl+a r` → Reload config

## Features

- Vi-style keys
- Mouse support
- 50k scrollback history
- Windows start at 1
